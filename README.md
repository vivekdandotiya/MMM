# IPL 2026 Profit Tracker ✨

A premium, Apple VisionOS-inspired web application to meticulously track betting profits and losses for the IPL 2026 season.

## Features
- **VisionOS Glassmorphism UI**: Stunning animated mesh gradients with deep frosted glass cards (`backdrop-filter`) and reactive neon glows.
- **Dynamic Live Dashboard**: Instantly calculates and visually displays Total Net P/L, Matches Played, and Win Rate.
- **Fail-Safe Logos**: Integrates reliable Wikimedia team assets with an automatic fallback mechanism generating clean UI-avatar initials if an image breaks.
- **Pagination & Filtering**: Client-side filtering by individual teams with strict 8-match-per-page pagination for zero clutter.
- **iOS Segment Controls**: Replaces clunky text fields with a seamless `Profit` / `Loss` segment toggle.

## Tech Stack
- Vanilla HTML5 
- Vanilla CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Local Storage, DOM Manipulation, Array Filtering)

## Getting Started
Simply open `index.html` in any modern web browser to start using the tracker. All data is saved directly in your browser's Local Storage, meaning it stays secure and private on your machine.
